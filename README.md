# The Fresnel Equations

一个交互式的菲涅耳方程教学 App（Streamlit 单文件）。展示光在介质界面上的**反射率/透射率**、**振幅系数与相位**、**布儒斯特角**、**全内反射与倏逝波**，并把背后的物理从麦克斯韦方程一路推到古斯-汉欣位移。

## 功能

- **反射率 R / 透射率 T** 随入射角的曲线，标注布儒斯特角 θ_B、临界角 θ_C。
- **振幅系数 r, t 与反射相位 φ**（可折叠）—— 展示功率图丢掉的符号/相位信息。
- **逐点查看**：拖动滑块看任意入射角下的光线示意图与系数；全内反射时实时显示
  - 倏逝波**穿透深度 δ**、**相对相移 Δφ**、**古斯-汉欣位移 D**；
  - r、t 的**复数值**（含全反射区 |r|=1 的相位）。
- **公式推导**（📖）：从麦克斯韦方程边界条件 → Fresnel 方程 → 布儒斯特角 / 临界角 → 倏逝波色散 → 古斯-汉欣位移的角谱（傅里叶）证明。
- 内置常用材料折射率预设，可下拉选择或手动输入。

## 运行

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

浏览器打开终端里给出的本地地址即可。

## 依赖

`streamlit`, `numpy`, `matplotlib`（见 `requirements.txt`）。
